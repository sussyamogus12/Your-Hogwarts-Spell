<h1>Ваше Заклинание в Хогвартс!</h1>
<a>Узнайте ваше заклинание из Гарри Поттера!</a>


<h2>Что делать:</h2>
1). <strong>Опишите</strong> себя / свой характер<br>
2). <strong>Нажмите</strong> "Наложить заклинание" или "Создать заклинание"<br>
3). <i>Подождите :3</i> <br>
4). ИИ <strong>сгенерирует</strong> результат!<br>
